Privacy Policy

Last updated: July 2026 — WhiteTree · Controller: Kan | Kan_Inthesky

WhiteTree is an anti-raid moderation bot for Discord. This page explains, simply, what data is processed, why, and what is never kept.

What the bot processes

  • Server configuration: protection thresholds, log channel, staff roles, whitelist, language.
  • Technical identifiers: server, channel, role and user IDs involved in a moderation action.
  • Sanctions log: who was sanctioned, when, by which module and why.
  • Active quarantines: the roles of a quarantined member, so they can be restored later.

Message content

Message content is analyzed in memory, in real time, to detect spam, invite links and abusive mentions. It is never stored on disk, never shared, and never used for any other purpose. Once checked, the content is discarded.

What the bot never does

  • No selling or sharing of data with third parties.
  • No advertising, no tracking.
  • No collection of direct messages or non-moderation data.

Retention and deletion

Configuration data is kept as long as the bot is on the server. When it is removed, administrators can request deletion of the server's data via the support contact below.

Security

Data is stored locally by the bot's host and protected by that host's access controls. The bot token and credentials are never exposed.

Your rights

You can request access to, or deletion of, the data associated with a server you administer by contacting the controller below.
